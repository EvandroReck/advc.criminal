📋 Sobre o SisDefesa

Este sistema foi desenvolvido como projeto didático para simular o ambiente real de um escritório de advocacia criminal. Ele conecta uma interface web moderna ao banco de dados SQLite, permitindo o cadastro, consulta e gerenciamento completo de informações sensíveis.

Funcionalidades avançadas como relacionamentos mestre-detalhe, autenticação de usuários e geração de relatórios estão implementadas para oferecer uma experiência profissional completa.

🛠️ Estrutura Tecnológica

  • Frontend: HTML5, CSS3 Moderno, JavaScript com Animações
  • Bibliotecas: Font Awesome, CSS Grid e Flexbox
  • Backend: Node.js com Express.js
  • Banco de Dados: SQLite com tabelas relacionais e segurança
  • Design: Temática profissional Preto & Dourado
  • Responsividade: Totalmente adaptado para desktop, tablet e mobile
